Método de Runge-Kutta 4to Orden

El método de Runge-Kutta de cuarto orden es una técnica para resolver ecuaciones diferenciales ordinarias. Este método tiene un error local de truncamiento de orden O(h⁵), lo que lo hace muy preciso para la mayoría de las aplicaciones.

Use la sintaxis de Python (x**2 para x², sin(x) para seno, etc.)
