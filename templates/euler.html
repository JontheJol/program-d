Método de Euler Mejorado

El método de Euler mejorado (también conocido como método de Heun) es un método numérico para resolver ecuaciones diferenciales ordinarias (EDO) de primer orden.

Use la sintaxis de Python (x**2 para x², sin(x) para seno, etc.)
